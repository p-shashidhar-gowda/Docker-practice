Tasks To Be Performed:
1. Create 5 custom container with 5 different default pages
2. Using Docker Compose deploy these 5 containers on port 81, 82, 83, 84
and 85 respectively

Solution docker-compose.yaml :

version: '3.8'
services:
  container1:
    build: ./container1
    ports:
      - "81:80"
  container2:
    build: ./container2
    ports:
      - "82:80"
  container3:
    build: ./container3
    ports:
      - "83:80"
  container4:
    build: ./container4
    ports:
      - "84:80"
  container5:
    build: ./container5
    ports:
      - "85:80"


to build :
docker compose up -d 
to stop :
docker compose down
